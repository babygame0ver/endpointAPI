# endpointAPI

Requirements

```
pip install -r require.txt
```

## Run endpointAPI :

```
python file.py
```

---

## Interact with endpointAPI :

```
curl -i http://127.0.0.1:5000/restapi
curl -i -H "Content-Type: application/json" -X POST -d '{"task_id":2}' http://127.0.0.1:5000/restapi/create
```
